similarity by regismesquita

A rails plugin intended to makes easy the models comparison and determine similarity between models or objects.

This project had been created with the objective of making easy and quickly to make comparison between objects , making easy the creation of programs using this kind of algorithm.

Dependencies

Some functionalities of this project had been designed to work with Active Record but it's not a dependence per se.

Install

Just download it to you rails-project/vendor/plugins and restart your rails server.
